## Инфраструктура Docker
### Дата: 19.03.2024
#### Описание: Настроена инфраструктура Docker для локальной разработки

#### Созданные файлы:
- `docker-compose.yml` - Основной файл для оркестрации Docker-контейнеров
- `backend/Dockerfile` - Конфигурация для Django бэкенда
- `frontend/Dockerfile` - Конфигурация для React фронтенда
- `docker/init-db.sql` - Скрипт инициализации базы данных

#### Основные сервисы:
- `db` - PostgreSQL база данных
- `backend` - Django REST API
- `frontend` - React приложение

#### Оптимизация для слабого компьютера:
- Использованы легковесные базовые образы:
  - `python:3.9-slim` для бэкенда
  - `node:16-alpine` для сборки фронтенда
  - `nginx:alpine` для фронтенда
  - `postgres:13-alpine` для базы данных
- Настроены тома для локальной разработки:
  - `postgres_data` для хранения данных PostgreSQL
  - Монтирование локальных директорий для hot-reload
  - Изолированный volume для node_modules
- Предусмотрена возможность запуска отдельных сервисов:
  - `docker-compose up backend` - только бэкенд и БД
  - `docker-compose up frontend` - только фронтенд
  - `docker-compose up db` - только база данных

#### Инструкция по запуску:
```bash
docker-compose up -d   # Запуск всех сервисов
docker-compose up backend   # Запуск только бэкенда и базы данных
docker-compose ps   # Проверка статуса контейнеров
docker-compose logs   # Просмотр логов
```
